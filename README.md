# 身体測定 プロジェクト
![実際の実装画面](./신체_측정_프로젝트_페이지.png)

## 身体測定プロジェクトとは？
Web環境上でリアルタイムのポーズ認識を活用し、
ストレートネック姿勢を角度ベースで測定・可視化する
身体測定プロジェクトです。

## プロジェクト概要
- 開発者 : キムミンギュ / 個人開発
- 開発人数 : 2024年12月 ～ 2025年4月（5か月）
- 使用技術 : Vue.js(Front-end), 
Express.js(Back-end)
MediaPipe Pose(姿勢分析アルゴリズム), 
MySQL(Data Base)

## プロジェクトの背景
パソコンの前で長時間座って勉強する友人たちの横顔を見て、
多くの人がストレートネック姿勢を維持したまま、
その変化に気づいていないケースが多いと感じました。
自身の前後の姿勢を撮影し、時間の経過による姿勢の変化を記録できれば、
姿勢改善に役立つのではないかと考えました。
これをWeb環境で実現し、画面の再読み込みなしに自然に画面が更新される
ユーザー体験を提供するため、
SPA構造のWebページとしてプロジェクトを進めることにしました。

## 主要機能
1. 身体状態の視覚化: ユーザーの首と肩の位置を自動で認識し、
画面上に線や角度で表示することで、
現在の状態を視覚的に把握できるように実装しました。
2. リアルタイム状態分析: 撮影中にリアルタイムで角度を計算し、
「安心」「注意」「警告」の状態を即座に判定し、
画面に表示する機能を実装しました。
3. 結果の自動保存と確認: 撮影終了後、分析結果は自動的に保存され、
ユーザーは日付ごとに過去の結果を
再確認することができます。
